# 🍕 Pizza Delivery App

A responsive Pizza Delivery web application built using React and Vite.

## 🚀 Features

- Modern and clean UI
- Fully responsive design
- Reusable React components
- Dynamic pizza listing
- Add-to-cart functionality
- Styled using CSS
- Fast development with Vite

## 🛠️ Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS3

## 📂 Project Structure

- Navbar
- Pizza Cards
- Footer
- Reusable Components
- Responsive Layout

## 📸 Preview

A simple and elegant pizza ordering interface designed for desktop and mobile devices.
